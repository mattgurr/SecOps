# 01 - Intake Questions - Cohort 3 - IT Professionals 

Questions for helpdesk / dev / sysadmin folks:

What is your current IT role (e.g., helpdesk, sysadmin, network engineer, developer, cloud engineer), and which parts of it do you enjoy most?

Which security‑adjacent work have you already touched in your role: account provisioning, patching, logging/monitoring, incident tickets, secure coding, cloud hardening?

Which security career directions appeal most: SOC / IR, cloud security, security engineering, AppSec, red team, GRC, or DFIR?

In the next 12–18 months, what would success look like: moving into a security‑titled role, becoming the security “go‑to” in your current team, or leading a security‑focused project?

What constraints do you have around changing roles: location, salary band, daytime availability, or current employer expectations?
